# 🎰 Jogo do Número Secreto

<img src = "img/Demonstração.gif" alt = "GIF da demonstração do site"/>

## 📰 Descrição

Primeiro projeto do **"Jogo do Número Secreto"**, desenvolvido durante o primeiro curso de Lógica de Programação da plataforma da Alura.

Esse projeto apresenta um jogo em que, ao inserir um número dentre os parâmetros já determinados (1 a 500, por exemplo), o sistema retorna se o "número secreto" é maior ou menor, fazendo com que você tenha que descobrir ele.
